序言

  • 本节主要是介绍 Markdown 语法的使用，方便后期记忆

标题

标题一对应的就是#标题
标题二对应的就是##标题
标题三对应的就是###标题
标题四对应的就是####标题
标题五对应的就是#####标题
标题六对应的就是######标题
标题一的字体最大，标题六的字体最小。一般情况下使用标题二

列表

列表里面分为 2 种
第一种是无序列表
第二种是有序列表

无序列表

  • 第一个无序列表(无序列表就是*加空格)
  • 第二个无序列表(无序列表就是*加空格)
  • 第三个无序列表(无需列表就是*加空格)

有序列表

  1. 有序列表第一个(就是序号加.加空格比如 1.+空格)
  2. 有序列表第二个(就是序号加.加空格比如 2.+空格)
  3. 有序列表第三个(就是序号加.加空格比如 3.+空格)

引用

这一段就是引用
可能颜色有点浅。但是后期能调。不过我习惯了.
引用的符号就是>后面加文字
引用要是想结束就是回车加换行

代码框

代码框的引入就是被(```)前后包起来即可。

<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>JS测试第一篇</title>
  </head>
  <body>
    <div id="div1" style="background:red;width:100px;height:100px"></div>
    <button id="btn1">点击我</button>
  </body>
</html>

图片


图片的格式就是 ![]()

调用的法则就是![图片的名称]+(图片的地址)
要是调用本地的图片就是(![图片名称]+(本地图片地址。要是不知道可以直接把图片拖拽到浏览器里))

链接

调用法则就是[链接的名称]+(链接的地址)他和图片比起来就少了一个感叹号

粗体和斜体

  • 粗体就是文字被 2 个星号包裹起来比如:粗体
  • 斜体就是文字被一个星号包裹起来斜体
  • 你要是又要粗体又要斜体只有文字
  • 让内容被 3 个星号包裹起来

Tab 用来缩进

一大段文字前面要是想缩进就可以按 Tab 键，这样直接就缩进了。还有边框。省时省力。 ##这个时候你要是需要一条分割线

三个星号表示的就是分割线。 ##以上就是 Markdown 最浅显的使用指南。
一共 11 个标签方便各位能够更好的记录

  3. 在 IOS 上 Ulysses for iPad
  4. Web 端请使用简书
